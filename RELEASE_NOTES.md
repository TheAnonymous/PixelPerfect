# PixelPerfect v1.0.0

Tiny UI. Big adventure.

The first stable PixelPerfect release contains 34 framework-neutral components, the handheld-green token system, self-hosted Pixelify Sans, a custom pixel icon sprite, and optional dependency-free behavior for tabs, dropdowns, dialogs, drawers, dismissible elements, and toasts.

Highlights:

- Semantic HTML and visible keyboard focus by default.
- Responsive layouts and reduced-motion handling.
- ESM and IIFE bundles with TypeScript declarations.
- Complete CSS, font, icon, license, documentation, and checksum assets.
- No runtime dependencies and no npm publication.

Verify the downloaded archive with the accompanying `.sha256` file, then use the archive’s internal `SHA256SUMS` to verify each packaged file.
