# PixelPerfect v2.1.0

The complete 34-component field guide is now one connected journey through Mossmere. Every chapter has its own code-native four-color diorama, every demo sits in a purposeful mini-composition, and the interactive examples behave like small real interface moments instead of isolated test stands.

Highlights:

- Nine connected destinations: Trailhead, Field Guide, Camp Command Post, Adventurer Registry, Overworld Map, Quest Log and Inventory, Checkpoint Shrine, Camp, and Journey's End.
- Local resettable scenes for quest commands, map scale, form validation, inventory selection and equip, repeatable checkpoint feedback, rest, and drawer actions.
- Exact Copy and View Code snippets for the currently visible demo markup.
- Explicit readable surfaces throughout inverted chapters, plus stable focus, hover, pressed, selected, disabled, loading, invalid, and open states.
- Complete static equivalents for reduced motion and a mobile presentation that keeps every component, state, and action.
- Expanded unit, cross-browser E2E, no-JS, interaction-isolation, and desktop/mobile visual coverage.

This is a nonbreaking showcase and quality release. The four-color palette, Pixelify Sans, semantic HTML, optional dependency-free JavaScript, zero runtime dependencies, all v2 classes and `data-pp-*` hooks, icon IDs, `init`, `destroy`, `toast`, `ToastOptions`, and `ToastHandle` are unchanged. No migration is required from v2.0.

Verify the downloaded archive with its `.sha256` sidecar, then use the internal `SHA256SUMS` file to verify every packaged file. PixelPerfect v2.1.0 is distributed through GitHub Releases and is not published to npm.
