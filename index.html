01

Quick start

Download the release, link one stylesheet, then add the optional script only when you need behavior.

Two files. Zero setup.

Ship a screen before the kettle boils.

PixelPerfect is framework-neutral. Copy the release assets into your project, use semantic HTML, and sprinkle in the pp- classes you need.

Download v2.0.0 ↗ 
<link rel="stylesheet" href="pixelperfect.css">

<button class="pp-button">Start quest</button>

<script src="pixelperfect.iife.js"></script>
<script>PixelPerfect.init();</script>
